## Overview

The `<sub>` and `<sup>` elements can be used to display the text as subscript and superscript, respectively.

![](https://i.imgur.com/J2ICIrX.png)


## Code Snippet

```html
// H2O
<p>H<sub>2</sub>O</p>

// (a + b)^2 = a^2 + b^2 + 2ab
<p>(a + b)<sup>2</sup> + a <sup>2</sup> + b <sup>2</sup> + 2ab
```
