## What is an extension?

> [!NOTE]
> A **filename extension**, **file name extension** or **file extension** is a suffix to the [name](https://en.wikipedia.org/wiki/Filename "Filename") of a [computer file](https://en.wikipedia.org/wiki/Computer_file "Computer file") (for example, `.txt`, `.docx`, `.md`). The extension indicates a characteristic of the file contents or its intended use. A filename extension is typically delimited from the rest of the filename with a [period](https://en.wikipedia.org/wiki/Full_stop "Full stop"), but in some systems[[1]](https://en.wikipedia.org/wiki/Filename_extension#cite_note-1) it is separated with spaces.

Some [file systems](https://en.wikipedia.org/wiki/File_system "File system") implement filename extensions as a feature of the file system itself and may limit the length and format of the extension, while others treat filename extensions as part of the filename without special distinction.

### General data formats[[edit](https://en.wikipedia.org/w/index.php?title=List_of_file_formats&action=edit&section=92 "Edit section: General data formats")]

These file formats are fairly well defined by long-term use or a general standard, but the content of each file is often highly specific to particular software or has been extended by further standards for specific uses.

#### Text-based[[edit](https://en.wikipedia.org/w/index.php?title=List_of_file_formats&action=edit&section=93 "Edit section: Text-based")]

- CSV – [comma-separated values](https://en.wikipedia.org/wiki/Comma-separated_values "Comma-separated values")
- HTML – [hyper text markup language](https://en.wikipedia.org/wiki/Hyper_text_markup_language "Hyper text markup language")
- CSS – [cascading style sheets](https://en.wikipedia.org/wiki/Cascading_style_sheets "Cascading style sheets")
- [INI](https://en.wikipedia.org/wiki/INI_file "INI file") – a configuration text file whose format is substantially similar between applications
- [JSON](https://en.wikipedia.org/wiki/JSON "JSON") – JavaScript Object Notation is an openly used data format now used by many languages, not just JavaScript
- TSV – [tab-separated values](https://en.wikipedia.org/wiki/Tab-separated_values "Tab-separated values")
- [XML](https://en.wikipedia.org/wiki/XML "XML") – an open data format
- [YAML](https://en.wikipedia.org/wiki/YAML "YAML") – an open data format
- [ReStructuredText](https://en.wikipedia.org/wiki/ReStructuredText "ReStructuredText") – an open text format for technical documents used mainly in the [Python](https://en.wikipedia.org/wiki/Python_(programming_language) "Python (programming language)") programming language
- .md – [Markdown](https://en.wikipedia.org/wiki/Markdown "Markdown") an open lightweight markup language to create simple but rich text, often used to format [README](https://en.wikipedia.org/wiki/README "README") files
- [AsciiDoc](https://en.wikipedia.org/wiki/AsciiDoc "AsciiDoc") – an open human-readable markup document format semantically equivalent to [DocBook](https://en.wikipedia.org/wiki/DocBook "DocBook")
- .yni – a configuration file similar to [YAML](https://en.wikipedia.org/wiki/YAML "YAML")

### Generic file extensions[[edit](https://en.wikipedia.org/w/index.php?title=List_of_file_formats&action=edit&section=94 "Edit section: Generic file extensions")]

These are filename extensions and broad types reused frequently with differing formats or no specific format by different programs.

#### Binary files[[edit](https://en.wikipedia.org/w/index.php?title=List_of_file_formats&action=edit&section=95 "Edit section: Binary files")]

- .bak, .bk – [Bak file](https://en.wikipedia.org/wiki/Bak_file "Bak file") various backup formats: some just copies of data files, some in application-specific data backup formats, some formats for general file backup programs
- BIN – binary data, often [memory dumps](https://en.wikipedia.org/wiki/Memory_dump "Memory dump") of executable code or data to be re-used by the same software that originated it
- DAT – data file, usually binary data proprietary to the program that created it, or an MPEG-1 stream of [Video CD](https://en.wikipedia.org/wiki/Video_CD "Video CD")
- DSK – file representations of various [disk storage](https://en.wikipedia.org/wiki/Disk_storage "Disk storage") images
- RAW – raw (unprocessed) data
- SZH – files that are associated with zero unique file types (the most prevalent being the Binary Data format)

#### Text files[[edit](https://en.wikipedia.org/w/index.php?title=List_of_file_formats&action=edit&section=96 "Edit section: Text files")]

- .cnf, .conf, .cfg – [configuration file](https://en.wikipedia.org/wiki/Configuration_file "Configuration file") substantially software-specific
- .log – [logfiles](https://en.wikipedia.org/wiki/Logfile "Logfile") usually text, but sometimes binary
- .asc, .text, .txt – human-readable [plain text](https://en.wikipedia.org/wiki/Plain_text "Plain text"), usually no more specific

### Partial files[[edit](https://en.wikipedia.org/w/index.php?title=List_of_file_formats&action=edit&section=97 "Edit section: Partial files")]

#### Differences and patches[[edit](https://en.wikipedia.org/w/index.php?title=List_of_file_formats&action=edit&section=98 "Edit section: Differences and patches")]

- [diff](https://en.wikipedia.org/wiki/Diff "Diff") – text file differences created by the program [diff](https://en.wikipedia.org/wiki/Diff "Diff") and applied as updates by [patch](https://en.wikipedia.org/wiki/Patch_(Unix) "Patch (Unix)")

#### Incomplete transfers[[edit](https://en.wikipedia.org/w/index.php?title=List_of_file_formats&action=edit&section=99 "Edit section: Incomplete transfers")]

- .!ut – !UT partly complete [uTorrent](https://en.wikipedia.org/wiki/UTorrent "UTorrent") download
- .crdownload – CRDOWNLOAD partly complete or incomplete [Google Chrome](https://en.wikipedia.org/wiki/Google_Chrome "Google Chrome") or [Microsoft Edge](https://en.wikipedia.org/wiki/Microsoft_Edge "Microsoft Edge") download
- .opdownload – OPDOWNLOAD partly complete or incomplete [Opera](https://en.wikipedia.org/wiki/Opera_(web_browser) "Opera (web browser)") download
- .part – PART partly complete [Mozilla Firefox](https://en.wikipedia.org/wiki/Mozilla_Firefox "Mozilla Firefox") or [Transmission](https://en.wikipedia.org/wiki/Transmission_(BitTorrent_client) "Transmission (BitTorrent client)") download
- .partial – PARTIAL partly complete [Internet Explorer](https://en.wikipedia.org/wiki/Internet_Explorer "Internet Explorer") or [Edge Legacy](https://en.wikipedia.org/wiki/Edge_Legacy "Edge Legacy") download

### Temporary files[[edit](https://en.wikipedia.org/w/index.php?title=List_of_file_formats&action=edit&section=100 "Edit section: Temporary files")]

- .temp, .tmp – [Temporary file](https://en.wikipedia.org/wiki/Temporary_file "Temporary file") sometimes in a specific format, but often just raw data in the middle of processing
- Pseudo-pipelines, Pseudo-pipeline – [Pseudo-pipeline](https://en.wikipedia.org/wiki/Pipeline_(software)#Pseudo-pipelines "Pipeline (software)") file used to simulate a software pipe
